# VoiceHub 文档

这是 [VoiceHub 智能点歌系统](https://github.com/laoshuikaixue/VoiceHub) 的官方文档仓库。

## 📖 关于 VoiceHub

VoiceHub 是一个现代化的智能点歌系统，支持多平台音乐搜索、智能排期管理，提供用户友好的界面设计。基于现代化技术栈构建，是一个稳定可靠的 Web 应用。

### 主要特性

- 🎵 **智能点歌**: 支持多平台音乐搜索和点歌
- 👥 **用户友好**: 直观的界面设计，简单易用
- 🏗️ **现代化架构**: 基于 Nuxt.js + Prisma + PostgreSQL

## 🏫 面向师生的功能

VoiceHub 专为校园广播站和师生设计，提供丰富的功能和便捷的操作体验。以下是系统的核心功能介绍：

### 🎵 点歌与音乐管理

- **多平台音乐搜索**: 支持网易云音乐、QQ音乐和哔哩哔哩搜索，满足不同音乐喜好
- **对唱模式**: 歌词页面提供「对唱模式」样式，让K歌体验更加专业和有趣
- **歌曲去重功能**: 智能识别重复歌曲，优化歌曲库管理，避免重复播放
- **音乐播放器**: 内置音乐播放器，支持进度控制和音质实时切换
- **音质切换**: 支持多种音质选择（标准、HQ、无损、Hi-Res等），动态获取最新播放链接
- **网易云音乐云盘上传**: 支持将音乐上传到网易云音乐云盘，音乐管理更灵活
- **网易云音乐上传功能**: 持续优化上传体验，确保音乐文件快速稳定上传

### 🔐 登录与用户管理

- **第三方登录**: 支持「GitHub/Google」登录方式（需绑定站内账号），提供更多登录选择
- **无密码登录**: 支持指纹/面容快捷登录（目前仅支持Windows hello/Passkey），更安全更方便
- **网易云音乐登录**: 支持扫码登录，登录后可搜索个人歌单、收藏及播客电台内容
- **一键添加到歌单**: 登录后支持将排期中的网易云音乐歌曲一键添加到个人歌单
- **用户管理**: 管理员添加用户，支持按年级班级分类
- **权限控制**: 多级权限管理，支持普通用户、管理员、超级管理员

### 📺 视频功能

- **B站视频剧集选择**: 新增B站视频剧集选择功能，观看更便捷
- **B站投稿视频跳转**: B站投稿添加视频跳转功能，内容互动更流畅
- **从歌单投稿**: 支持从个人歌单中直接投稿歌曲到系统
- **从最近播放投稿**: 支持从最近播放记录中投稿歌曲
- **播客电台投稿**: 支持搜索和投稿播客电台内容

### 📅 排期与管理

- **排期管理**: 管理员可以通过拖拽界面进行歌曲排期和顺序管理
- **排期草稿**: 支持保存排期草稿功能，允许管理员分步完成排期安排
- **播出时段**: 灵活配置播出时段，支持多时段管理
- **打印排期**: 支持自定义纸张大小、内容选择、编写备注和PDF导出的打印功能
- **学期管理**: 管理员可设置当前学期，自动关联点歌记录
- **公开展示**: 公开展示歌曲播放排期，按日期分组展示
- **投稿限额管理**: 灵活配置用户投稿限制，支持按时间段、用户角色设置不同的投稿额度

### 🔔 通知与互动

- **实时通知**: 歌曲被选中、投票和系统通知
- **通知设置**: 用户可自定义通知偏好，支持独立页面设置
- **批量通知**: 管理员可向特定用户群体发送通知
- **社交账号绑定**: 支持绑定外部账号，同步推送通知到外部平台

### 🎨 用户体验

- **现代UI**: 响应式设计，深色主题，流畅的动画效果
- **玻璃态设计**: 现代化的视觉效果和交互体验
- **交互反馈**: hover效果，点击反馈，状态变化动画
- **移动端优化**: 适配支持移动设备访问，触摸友好的交互设计

## 🚀 本地开发

### 环境要求

- Node.js 18.0 或更高版本
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

访问 [http://localhost:3000](http://localhost:3000) 查看文档站点。

### 构建生产版本

```bash
npm run build
```
