# 常见问题解决方案

本文档收集了华青之声点歌台使用过程中的常见问题及其解决方案。

## 账户相关问题

### 1. 忘记密码怎么办？

**问题描述**：
忘记了登录密码，无法登录点歌台。

**解决方案**：
1. 打开微信，搜索 **hw student union** 微信公众号
2. 输入 **我要点歌** 进行咨询
3. 按照公众号提示重置密码

### 2. 如何修改密码？

**问题描述**：
想修改当前的登录密码。

**解决方案**：
1. 登录点歌台
2. 点击页面右上角的用户头像
3. 选择"修改密码"
4. 输入旧密码和新密码
5. 点击"保存"完成修改

### 3. 无法登录怎么办？

**问题描述**：
输入账号密码后无法登录，提示错误。

**解决方案**：
1. 检查账号和密码是否输入正确（注意大小写）
2. 确认您的账号是否已经激活
3. 如仍无法登录，请联系微信公众号 **hw student union** 咨询

## 点歌相关问题

### 1. 如何点歌？

**问题描述**：
不知道如何提交点歌请求。

**解决方案**：
1. 登录点歌台
2. 在首页搜索框中输入歌曲名称或歌手名
3. 点击搜索结果中的歌曲
4. 确认点歌信息，点击"点歌"按钮
5. 等待广播站审核播放

### 2. 为什么我的点歌没有出现在排期中？

**问题描述**：
提交了点歌请求，但在排期中看不到。

**解决方案**：
1. 点歌需要广播站审核，请耐心等待
2. 检查歌曲是否符合广播站的播放规定
3. 确认点歌数量是否已达上限

### 3. 可以点歌多少首？

**问题描述**：
想知道最多可以同时点多少首歌。

**解决方案**：
系统有点歌数量限制，具体限制请关注微信公众号 **hw student union** 咨询。

### 4. 为什么搜索不到某些歌曲？

**问题描述**：
输入歌曲名称或歌手名，但搜索不到结果。

**解决方案**：
1. 可能是版权限制或歌曲不在支持的平台上
2. 尝试使用不同的关键词搜索
3. 检查输入是否正确

## 投票相关问题

### 1. 如何为歌曲投票？

**问题描述**：
想为喜欢的歌曲投票，但不知道如何操作。

**解决方案**：
1. 登录点歌台
2. 在歌曲列表中找到喜欢的歌曲
3. 点击"投票"按钮
4. 投票成功后会显示投票状态

### 2. 可以为同一首歌多次投票吗？

**问题描述**：
想为同一首歌多次投票。

**解决方案**：
每首歌曲每个用户只能投票一次，系统会自动限制。

### 3. 投票对歌曲播放有什么影响？

**问题描述**：
想知道投票是否会影响歌曲的播放顺序。

**解决方案**：
投票可以提高歌曲的热度，热度高的歌曲有更高的播放优先级，但最终播放顺序由广播站决定。

## 排期相关问题

### 1. 如何查看播放排期？

**问题描述**：
想查看近期的广播播放排期。

**解决方案**：
1. 登录点歌台
2. 点击导航栏中的"排期管理"或"节目单"
3. 选择日期查看相应的排期

### 2. 排期会随时变化吗？

**问题描述**：
想知道排期是否会临时调整。

**解决方案**：
排期可能会根据实际情况进行调整，建议您关注最新的排期信息。

## 广播站管理问题

### 1. 如何审核点歌请求？

**问题描述**：
广播站成员不知道如何审核师生提交的点歌请求。

**解决方案**：
1. 以管理员身份登录点歌台
2. 进入管理后台
3. 点击"歌曲管理"模块
4. 查看待审核歌曲列表
5. 审核歌曲内容，点击"通过"或"拒绝"

### 2. 如何进行排期管理？

**问题描述**：
广播站成员不知道如何安排歌曲播放顺序。

**解决方案**：
1. 以管理员身份登录点歌台
2. 进入管理后台
3. 点击"排期管理"模块
4. 选择日期和播放时段
5. 从左侧"待排歌曲"中拖拽歌曲到右侧"播放顺序"列表
6. 调整播放顺序，点击"保存"

### 3. 如何发送系统通知？

**问题描述**：
想向师生发送系统通知。

**解决方案**：
1. 以管理员身份登录点歌台
2. 进入管理后台
3. 点击"通知管理"模块
4. 点击"发送通知"
5. 选择接收者和通知类型
6. 编写通知内容，点击"发送"
