# 华青之声点歌台介绍

欢迎使用 **华青之声点歌台** - 华北水利水电大学校园广播站点歌系统！

## 什么是华青之声点歌台？

华青之声点歌台是华北水利水电大学校园广播站专属的点歌系统，为全校师生提供便捷的点歌、投票服务，同时为广播站提供高效的排期管理和播放功能。

## 主要特性

### 🎵 智能点歌系统
- **多平台音乐搜索**：支持网易云音乐和QQ音乐搜索
- **投票机制**：用户可以为喜欢的歌曲投票
- **防重复播放**：智能避免重复播放相同歌曲

### 👥 用户管理系统
- **多级权限控制**：支持普通用户（师生）、管理员（广播站成员）、超级管理员（广播站站长）
- **便捷登录**：使用校园统一身份认证

### 📅 排期管理
- **拖拽式排期**：直观的拖拽界面进行歌曲排期
- **多时段支持**：支持配置多个播出时段
- **学期管理**：按学期组织点歌记录
