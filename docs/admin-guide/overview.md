# 广播站管理指南概览

欢迎使用华青之声点歌台管理后台！本指南将帮助您了解如何使用系统的管理功能。

## 管理后台概览

### 访问管理后台

使用广播站管理员账户登录后，您可以通过以下方式访问管理功能：

1. 点击右上角的用户菜单
2. 选择"管理后台"

### 角色说明

华青之声点歌台管理分为两个角色：

- **管理员**：广播站成员，负责歌曲审核、排期管理等
- **超级管理员**：广播站站长，拥有全部管理权限，包括用户管理和系统配置

### 管理界面布局

管理后台采用标签页设计，包含以下主要模块：

- **概览仪表盘**：系统统计和快速操作
- **排期管理**：歌曲排期和播放顺序管理
- **歌曲管理**：歌曲库管理和审核
- **通知管理**：系统通知发送和管理

## 核心管理功能

### 📊 概览仪表盘

提供系统运行状态的全面概览：

- **统计数据**：用户数量、歌曲数量、点歌统计
- **最近活动**：最新的点歌、投票和用户活动
- **快速操作**：常用管理操作的快捷入口

### 📅 排期管理

华青之声点歌台的核心功能，支持直观的拖拽式排期：

**主要特性**：
- **拖拽排期**：从左侧待排歌曲拖拽到右侧播放顺序
- **顺序调整**：在播放列表中拖拽调整播放顺序
- **时段管理**：按播放时段组织排期

**操作流程**：
1. 选择日期和播放时段
2. 从左侧"待排歌曲"中选择歌曲
3. 拖拽到右侧"播放顺序"列表
4. 调整播放顺序
5. 保存排期

### 🎵 歌曲管理

管理歌曲库和审核点歌请求：

**主要功能**：
- **歌曲审核**：审核师生提交的点歌请求
- **歌曲库管理**：管理系统中的歌曲信息
- **重复检查**：避免重复播放相同歌曲

**审核流程**：
1. 进入"歌曲管理"模块
2. 查看待审核歌曲列表
3. 审核歌曲内容（确保符合广播站规定）
4. 通过或拒绝点歌请求
5. 对于通过的歌曲，可添加到待排歌曲列表

### 🔔 通知管理

发送和管理系统通知：

**通知发送**：
- **选择接收者**：全体用户、特定年级、班级
- **编写消息**：自定义通知内容
- **发送历史**：查看历史通知记录

## 管理规范

作为广播站管理员，请遵守以下管理规范：

1. **公正审核**：对所有点歌请求进行公正审核
2. **内容合规**：确保播放歌曲符合校园文化和广播站规定
3. **及时处理**：及时审核点歌请求，保持良好的用户体验
4. **合理排期**：根据广播时段合理安排歌曲播放顺序
5. **安全操作**：妥善保管管理账户，定期修改密码
