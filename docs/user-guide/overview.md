# 师生使用指南概览

欢迎使用华青之声点歌台！本指南将帮助您了解如何使用点歌台的各项功能。

## 界面概览

### 主页面

华青之声点歌台的主页面包含以下主要区域：

- **导航栏**：包含华青之声Logo、导航菜单和用户操作
- **排期展示**：显示当前和未来的广播播放排期
- **快速操作**：点歌、投票等快捷入口
- **通知中心**：显示系统通知和消息

## 主要功能

### 🎵 点歌功能

- **音乐搜索**：支持网易云音乐和QQ音乐搜索
- **点歌提交**：提交点歌请求等待广播站审核

### 🗳️ 投票功能

- **歌曲投票**：为喜欢的歌曲投票
- **热度排序**：根据投票数量排序歌曲

### 📅 排期查看

- **当前排期**：查看今日播放安排
- **未来排期**：查看未来几天的播放计划
- **时段分组**：按播放时段分组显示

## 快速开始

### 1. 获取账户

1. 打开微信，搜索 **hw student union** 微信公众号
2. 输入 **我要点歌** 进行咨询
3. 按照公众号提示获取您的点歌账号和密码

### 2. 登录系统

1. 访问 [华青之声点歌台](https://www.hwfm.online/)
2. 点击右上角的"登录"按钮
3. 输入您的账号和密码
4. 登录成功后，点击"修改密码"更新初始密码

### 3. 提交点歌

1. 在首页搜索框中输入您想点的歌曲名称或歌手名
2. 点击搜索结果中的歌曲
3. 确认点歌信息，点击"点歌"按钮
4. 等待广播站审核播放

### 4. 参与投票

1. 在歌曲列表中找到喜欢的歌曲
2. 点击"投票"按钮
3. 查看投票结果和热度排名

## 使用技巧

- **使用关键词搜索**：输入歌名、歌手或专辑名
- **避免重复点歌**：检查是否已有相同歌曲

### 智能投票

- **支持喜欢的歌曲**：为喜欢的歌曲投票，提高播放优先级

## 常见问题

### 点歌相关

**Q: 为什么我的点歌没有出现在排期中？**
A: 点歌需要广播站审核，请耐心等待。

**Q: 可以点歌多少首？**
A: 系统有点歌数量限制，具体限制请关注微信公众号咨询。

**Q: 为什么搜索不到某些歌曲？**
A: 可能是版权限制或歌曲不在支持的平台上，建议尝试不同的搜索关键词。

### 账户相关

**Q: 忘记密码怎么办？**
A: 关注微信公众号 **hw student union**，输入 **我要点歌** 进行咨询。

**Q: 如何修改密码？**
A: 登录后，点击页面右上角的"修改密码"按钮即可修改。
