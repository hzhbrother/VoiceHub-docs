# 首次操作指南

欢迎使用华青之声点歌台！本指南将帮助您完成首次使用的基本操作。

## 访问点歌台

1. 打开浏览器，访问华青之声点歌台官方地址：
   [https://www.hwfm.online/](https://www.hwfm.online/)

2. 您将看到华青之声点歌台的首页

## 登录点歌台

### 1. 获取登录账号

首次使用前，您需要获取登录账号和密码：

- 打开微信，搜索 **hw student union** 微信公众号
- 输入 **我要点歌** 进行咨询
- 按照公众号提示获取您的点歌账号和密码

### 2. 登录操作

1. 在点歌台首页点击右上角的"登录"按钮
2. 输入您从公众号获取的账号和密码
3. 点击"登录"按钮进入点歌台

## 基本操作

### 1. 修改密码

登录成功后，您可以修改初始密码：

1. 点击页面右上角的"修改密码"按钮
2. 输入旧密码和新密码
3. 点击"保存"按钮完成修改

### 2. 点歌操作

1. 在首页搜索框中输入您想点的歌曲名称或歌手名
2. 点击搜索结果中的歌曲
3. 确认点歌信息，点击"点歌"按钮
4. 等待广播站审核播放

### 3. 查看排期

1. 点击导航栏中的"排期管理"或"节目单"
2. 查看近期的广播节目排期
3. 您可以看到已点歌曲的播放时间

## 角色说明

华青之声点歌台分为三个角色：

- **普通用户**：全校师生，可进行点歌、投票等操作
- **管理员**：广播站成员，负责歌曲审核、排期管理等
- **超级管理员**：广播站站长，拥有全部管理权限

## 获取帮助

如果您在使用过程中遇到任何问题：

1. 检查本使用指南
2. 关注微信公众号 **hw student union**
3. 输入 **我要点歌** 进行咨询
