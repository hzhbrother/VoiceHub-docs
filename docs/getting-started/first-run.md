# 首次操作指南

欢迎使用华青之声点歌台！本指南将帮助您完成首次使用的基本操作。

## 访问点歌台

1. 打开浏览器，访问华青之声点歌台官方地址：
   [https://www.hwfm.online/](https://www.hwfm.online/)

2. 您将看到华青之声点歌台的首页

## 登录点歌台

### 1. 获取登录账号

首次使用前，您需要获取登录账号和密码：

- 打开微信，搜索 **hw student union** 微信公众号
- 输入 **我要点歌** 进行咨询
- 按照公众号提示获取您的点歌账号和密码

### 2. 登录操作

1. 在点歌台首页点击右上角的"登录"按钮
2. 输入您从公众号获取的账号和密码
3. 点击"登录"按钮进入点歌台

## 基本操作

### 1. 修改密码

登录成功后，您可以修改初始密码：

1. 点击页面右上角的"修改密码"按钮
2. 输入旧密码和新密码
3. 点击"保存"按钮完成修改

### 2. 点歌操作

1. 在首页搜索框中输入您想点的歌曲名称或歌手名
2. 点击搜索结果中的歌曲
3. 确认点歌信息，点击"点歌"按钮
4. 等待广播站审核播放

### 3. 查看排期

1. 点击导航栏中的"排期管理"或"节目单"
2. 查看近期的广播节目排期
3. 您可以看到已点歌曲的播放时间

## 系统功能介绍

华青之声点歌台提供丰富的功能，为您带来便捷的点歌体验：

### 🎵 点歌与音乐管理

- **多平台音乐搜索**：支持网易云音乐、QQ音乐和哔哩哔哩搜索，满足不同音乐喜好
- **对唱模式**：歌词页面提供「对唱模式」样式，让K歌体验更加专业和有趣
- **音质切换**：支持多种音质选择（标准、HQ、无损、Hi-Res等）
- **网易云音乐云盘上传**：支持将音乐上传到网易云音乐云盘，音乐管理更灵活

### 🔐 登录方式

- **账号密码登录**：使用从公众号获取的账号密码登录
- **第三方登录**：支持「GitHub/Google」登录方式（需绑定站内账号）
- **无密码登录**：支持指纹/面容快捷登录（目前仅支持Windows hello/Passkey），更安全更方便
- **网易云音乐登录**：支持扫码登录，登录后可搜索个人歌单、收藏及播客电台内容

### 📺 视频功能

- **B站视频剧集选择**：支持选择B站视频剧集，观看更便捷
- **B站投稿视频跳转**：B站投稿添加视频跳转功能，内容互动更流畅

### 📅 排期与管理

- **排期查看**：公开展示歌曲播放排期，按日期分组展示
- **投票功能**：支持为喜欢的歌曲投票，增加播放机会

## 角色说明

华青之声点歌台分为三个角色：

- **普通用户**：全校师生，可进行点歌、投票等操作
- **管理员**：广播站成员，负责歌曲审核、排期管理等
- **超级管理员**：广播站站长，拥有全部管理权限
